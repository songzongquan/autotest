# autotest
autotest是一个用python写的命令行小工具，它实现了将jira+zephyr 和 selenium 集成起来的效果。
统一在jira中进行测试用例与计划的管理、测试结果的统计与分析，不管是手动测试还是自动化测试脚本的测试。
